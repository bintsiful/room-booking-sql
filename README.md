This project demonstrates 4 different approaches to solving room/bed availability problems in Oracle SQL:

1. Basic overlap logic
2. Daily occupancy expansion
3. Interval boundary optimization
4. Proper data modeling with explicit beds

It highlights the trade-offs between simplicity, accuracy, and performance.
